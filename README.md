## How to run this

cd ECE2310Project

dotnet build "{Your directory to this file}\ECE2310Project.csproj"
{Your directory to this file}\ECE2310Project\ECE2310Project\bin\Debug\ECE2310Project.exe

### What you will see
    * The planner opens up, you are able to choose a date and time by interacting with the ui.
    * Choose the date, time, name and description of the event
    * When time reaches, it pops a small notification window from bottom right
